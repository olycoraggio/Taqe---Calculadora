Projeto Taqe

Desenvolvedor:
Orlando Vieira Lopes

App:
Calculadora simples para operações de:

-Adição;
-Subtração;
-Multiplicação;
-Divisão.

Use:
Para iniciar uma operação é necessário digita-la, e logo ela irá aparecer no primeiro campo. Ao pressionar o botão de igualdade, 
o resultado da operação ira aparecer no segundo campo.
Para realizar outra operação, é necessário limpar o campo de resultado pressionando o botão Del.
